# Homebrew Tap for Vibe CLI

This tap provides the `vibe` formula for installing the Vibe CLI.

## Installation

First, add this tap:

```bash
brew tap <your-github-username>/vibe
```

Then, install the formula:

```bash
brew install vibe
```

## Usage

After installation, you can run:

```bash
vibe --help
```

## Development

To update the formula, you can edit the `vibe.rb` file in this directory.
